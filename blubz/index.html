What is Blubz?

Blubz is an interactive journal application built for Android OS. The application notifies the user daily to post either a 140 character text or a picture to be input into the Blub database. Every Monday another notification is presented to the user to check a post from their past. Preferences for notification can be changed in the applications setting menu located in the action bar.

How to Download

You can find our most up to date application at the Google Play Store by searching for Blubz.

Types of Blubz

Text Input: The text input is restricted to a 140 character message.

Photo Input: The photo input is limited to one photograph taken by the phone.
